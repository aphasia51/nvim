## ✅About
Neovim configuration written in lua

## 💥Features
- 👻 Less than 50 plugins
- 🏂 The start speed is fast(about 50ms)

### 👹Install
```
git clone https://github.com/xiahuas/nvim.git ~/.config/nvim
```

### Something
In my opinion, neovim (vim) is a powerful editor. After you are familiar with its various shortcut keys, you can refer to the repositories of some big guys to write your own configuration. You'll install a variety of plugins to suit your development needs, then explore how to improve neovim's speed. Any way, I hope you can configure a neovim that you can use easily.
